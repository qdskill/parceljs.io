### 配置本地运行

1. 克隆仓库

	```
	git clone git@github.com:{Your Github Name}/parceljs.io.git
	cd parceljs.io
	```
2. 安装依赖

	```
	npm install
	```
3. 编译
	
	```
	npm run build
	```
4. 运行
	
	```
	npm start
	```
